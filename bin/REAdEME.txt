<ul>
<li>Runs on any standard x86 hardware</li>
</ul>
<ul>
<li>Supports 64-bit guest operating systems, including Windows, Linux, and SolarisCan be managed by VMware VirtualCenter to efficiently provision, monitor and manage infrastructure from a central management console</li>
</ul>
<ul>
<li>Supports two-processor Virtual SMP, enabling a single virtual machine to span multiple physical processors</li>
</ul>
<ul>
<li>Runs on a wider variety of Linux and Windows host and guest operating systems than any server virtualization product on the market Captures entire state of a virtual machine and rolls back at any time with the click of a single button</li>
</ul>
<ul>
<li>Installs like an application, with quick and easy, wizard-driven installation</li>
</ul>
<ul>
<li>Quick and easy, wizard-driven virtual machine creation</li>
</ul>
<ul>
<li>Opens VMware or Microsoft virtual machine format and Symantec LiveState Recovery images with VM Importer</li>
</ul>
<ul>
<li>Supports Intel Virtualization Technology</li>
</ul>
<ul>
<li>Protects investment with an easy upgrade path to VMware Infrastructure</li>
</ul>
<p><strong>WARNING:</strong> &#8220;this tutorial is meant for users that have a good knowledge of development tools and manual installation process and will be hardly supported by the Ubuntu community. Standard supported procedure are to install packages from the official repositories, not to compile them by  hand&#8221;.<br />
<strong>Install Vmware Server in Ubuntu 7.10 Gutsy Gibbon </strong></p>
<p><strong>Preparing your system</strong></p>
<p>First you need to install the following packages</p>
<p>sudo apt-get install build-essential linux-headers-`uname -r`</p>
<p>In my case i have used the following command</p>
<p>sudo apt-get install build-essential linux-headers-2.6.22-14-generic</p>
<p>sudo apt-get install xinetd</p>
<p>Download Vmware server</p>
<p>First you need to register from <a href="http://register.vmware.com/content/registration.html" target="_blank">here</a> to downlaod and to get registration number</p>
<p>Create a directory</p>
<p>mkdir vmware</p>
<p>cd vmware</p>
<p>Download vmware sorce code using the following comamnd in to</p>
<p>wget http://download3.vmware.com/software/vmserver/VMware-server-1.0.4-56528.tar.gz</p>
<p>Extract the file using the following command</p>
<p>tar -vxzf VMware-server-1.0.4-56528.tar.gz</p>
<p>Now you have vmware-server-distrib directory</p>
<p>cd vmware-server-distrib</p>
<p>Start the installation using the following comamnd</p>
<p>sudo ./vmware-install.pl</p>
<p>This will start the installation you can see the steps in the following screens you can accept all defaults</p>
<p><img src="http://www.ubuntugeek.com/images/gvm/1.jpg" alt="" align="middle" /></p>
<p><img src="http://www.ubuntugeek.com/images/gvm/2.jpg" alt="" align="middle" /></p>
<p><img src="http://www.ubuntugeek.com/images/gvm/3.jpg" alt="" align="middle" /><br />
<img src="http://www.ubuntugeek.com/images/gvm/4.jpg" alt="" align="middle" /></p>
<p>This will complete the instalaltion</p>
<p><strong>Using Vmware Server in Ubuntu</strong></p>
<p>If you want to open Vmware Server Console go to Applications—&gt;System Tools—&gt;Vmware Server Console</p>
<p><img src="http://www.ubuntugeek.com/images/gvm/5.jpg" alt="" align="middle" /></p>
<p>Once it Opens you should see similar to the following selct localhost option and click on connect</p>
<p><img src="http://www.ubuntugeek.com/images/gvm/6.jpg" alt="" align="middle" /></p>
<p>Now you should see similar to the following screen if you want to create new virtual machine click on “Create a new virtual machine”</p>
<p><img src="http://www.ubuntugeek.com/images/gvm/7.jpg" alt="" align="middle" /></p>
<p>Once it open you should see the following screen and click on Next</p>
<p><img src="http://www.ubuntugeek.com/images/gvm/8.jpg" alt="" align="middle" /></p>
<p>Virtual machine configuration options select Typical click on Next</p>
<p><img src="http://www.ubuntugeek.com/images/gvm/9.jpg" alt="" align="middle" /></p>
<p>Select Guest operating system click on Next</p>
<p><img src="http://www.ubuntugeek.com/images/gvm/10.jpg" alt="" align="middle" /></p>
<p>You want to change the guest operating system name and location of this GuestOS you can do here click on Next</p>
<p><img src="http://www.ubuntugeek.com/images/gvm/11.jpg" alt="" align="middle" /></p>
<p>Select Network connection Option and Click on Next</p>
<p><img src="http://www.ubuntugeek.com/images/gvm/12.jpg" alt="" align="middle" /></p>
<p>Select your disk size and click on Finish</p>
<p><img src="http://www.ubuntugeek.com/images/gvm/13.jpg" alt="" align="middle" /></p>
<p>Vmware Disk Creation in Progress</p>
<p><img src="http://www.ubuntugeek.com/images/gvm/14.jpg" alt="" align="middle" /></p>
<p>Once it craeted new disk your machine is ready to install vmware GuestOS</p>
<p><img src="http://www.ubuntugeek.com/images/gvm/15.jpg" alt="" align="middle" /></p>
<p>Vmware-Server Version Details</p>
<p><img src="http://www.ubuntugeek.com/images/gvm/16.jpg" alt="" align="middle" /></p>

<p>Run the following command from your terminal</p>
<blockquote><p>sudo vmware-uninstall.pl</p></blockquote>

            count = parseInt(jQuery.cookie('wwsgd_visits'), 10) + 1;
        }
        jQuery.cookie('wwsgd_visits', count, { expires: 365, path: "/" });

        if ( count <= 0 ) {
            jQuery(".wwsgd").show();
        }
    });
